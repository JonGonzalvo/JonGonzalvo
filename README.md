## About Me

I'm a QA Engineer who broke into tech without a CS degree — and now I help others do the same.

I spent years learning quality assurance the hard way: scattered resources, outdated tutorials, and figuring things out by doing. So I wrote the guide I wish I'd had when I started.

- 🔍 I specialize in **manual testing, API testing, and test automation**
- 📖 I wrote **[Quality Assurance 101: Zero to Hero](https://qa101book.com)** — a complete beginner's guide to landing your first QA role
- 🌎 Based in the Dominican Republic, working in global tech
