# Test Cases

This folder contains test cases written for the demo e-commerce application.

## Structure

| File | Module | # of Test Cases |
|---|---|---|
| `ecommerce-test-cases.md` | Login, Cart, Checkout, Search, Contact | 25+ |

## How These Were Written

Each test case follows this format:
- **TC ID** — Unique identifier (TC-001, TC-002…)
- **Title** — What the test verifies, in plain English
- **Preconditions** — What must be true before execution
- **Steps** — Numbered, specific, reproducible
- **Expected Result** — Exactly what should happen
- **Status** — Pass / Fail / Blocked / N/A
